Our Approach

Discover how our machine learning platform can transform the way you prevent depression and anxiety among patients.

Data Selection

For the data selection, we conducted an exploration of the surveys conducted by the National Institute of Statistics until we came across one from 2020 containing data on numerous aspects of the respondents. We deemed it appropriate to utilize these observations to predict depression and/or anxiety among the users as the target variables.

2

Data Cleaning

During the data cleaning process, our focus is on preparing the data for analysis. This involves formatting the data into types that are compatible with our analysis tools and methods. Additionally, we identify and remove variables that contain a large number of missing values. This ensures that our dataset is consistent, complete, and suitable for further analysis.

3

Exploratory Data Analysis

During Exploratory Data Analysis (EDA), we conduct both univariate and multivariate analysis to uncover patterns and relationships within the data. This includes attempting to identify correlations between variables and aggregating variables to simplify the analysis process. Ultimately, we perform a split of the dataset into training and testing sets to facilitate model development and evaluation.

Notebook
4

Machine Learning

For the machine learning model, we employ the XGBoost algorithm and iterate using the SelectKBest method to determine the minimum number of variables necessary without sacrificing too much accuracy. We train the model and optimize it by fine-tuning the hyperparameters through a random search process. This iterative approach helps us find the optimal configuration for our model, ensuring that it achieves the best possible performance on the given task.

Elevating Precision with XGBoost

This boosting algorithm seamlessly blends superior accuracy, computational efficiency, advanced modeling, and unmatched flexibility.

  • Handles diverse data types effortlessly, requiring minimal preprocessing
  • Captures intricate non-linear relationships for enhanced modeling
  • Allows regularization of features, particularly beneficial for datasets with numerous variables
  • Maintains computational efficiency without the high demands of other complex models like neural networks

Model diversity is a key factor for the good performance of an ensemble.
